# Automated Exam and Individual Task Generator

Программа на для автоматической генерации:

- **индивидуальных заданий для студентов** (Word + Excel)
- **экзаменационных билетов** (PDF)

Система считывает настройки и входные данные из **конфигурационного Excel-файла** и формирует отформатированные документы для использования в учебном процессе.

Проект предназначен для упрощения подготовки:

- коллоквиумов
- экзаменов
- индивидуальных заданий
- алгоритмических задач для студентов

---

# Основные возможности

## Режим индивидуальных заданий

Генерация индивидуальных вариантов заданий для студентов.

Результат:

- **Excel-таблица** со сгенерированными вариантами
- **Word-документ** с оформленными листами заданий

---

## Экзаменационный режим

Генерация экзаменационных билетов.

Результат:

- **PDF-документ** с оформленными листами заданий
  
---
Все параметры программы задаются в файле **config.xlsx**
